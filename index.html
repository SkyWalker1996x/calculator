Для вычисления квадратного корня из числа, сначала необходимо ввести число, потом операцию извлечения квадратного корня √ после чего будет сразу выдан результат

Для вычисления дробных числе меньше нуля рекомендуется использовать следующую запись: 0.2 + 0.1
